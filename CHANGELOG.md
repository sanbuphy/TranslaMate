### Added
- Initial release of TranslaMate
- AI-powered translation using OpenAI-compatible APIs
- Support for 20+ languages
- Translation history with local storage
- Dark mode support
- Cross-platform support (Windows, macOS)
- Global hotkey (Ctrl/Cmd + Shift + T)
- Beautiful and modern UI
- Copy to clipboard functionality
- Language swap feature
- Configurable API settings (base URL, model, temperature, max tokens)
